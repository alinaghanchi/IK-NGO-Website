# InAmigos Foundation NGO Website

A modern and responsive NGO website developed for **InAmigos Foundation** using React JS and Vite.

## 🌟 Live Demo

https://ik-ngo-website.vercel.app/

## 📂 GitHub Repository

https://github.com/alinaghanchi/IK-NGO-Website
## 📖 About the Project

The InAmigos Foundation website was created to showcase the NGO's mission, impact, and community initiatives. The platform highlights educational programs, environmental activities, women empowerment initiatives, animal welfare projects, and social development campaigns.

## 🚀 Features

* Responsive Design
* Modern Navigation Bar
* Hero Section
* About NGO Section
* Impact Statistics
* Project Showcase
* Image Gallery
* Volunteer Section
* Contact Form
* Professional Footer
* Mobile-Friendly Interface

## 🛠️ Technologies Used

* React JS
* Vite
* HTML5
* CSS3
* JavaScript
* React Icons
* GitHub
* Netlify


## 🎯 Purpose

To provide an engaging digital presence for InAmigos Foundation and help visitors learn about the NGO's activities, projects, and impact.
